## `$ prisma1 admin`

Opens a [Prisma Admin](el3e) for the current service. 

### Usage

```sh
prisma1 admin [flags]
```

### Flags

```
 -e, --env-file ENV-FILE    Path to .env file to inject env vars
```

### Examples

**Open Prisma Admin**

```sh
prisma1 admin
```
