## FAQ

[Prisma Cloud](https://app.prisma.io) is running on AWS Lambda in `eu-west-1` region. To learn which IP addresses are used in that region, you can check out the [AWS docs](https://docs.aws.amazon.com/general/latest/gr/aws-ip-ranges.html) on that topic.
